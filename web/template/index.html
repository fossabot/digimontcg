Digimon TCG Online Database

An unofficial API for the Digimon Trading Card Game

Your digital gate for digimon card information

From Alphamon to Zubamon, all digimon cards in the English format (Ver.1.0 to Draconic Roar) are available, including alternate artwork.

Get access to card names, details, and alternate artwork

Includes basic card info as well as effects, digivolution requirements, and other useful info.

Links to high resolution artwork (including alternate art) is included for each card.

This project can be used as the basis for other Digimon TCG related projects.
